# Custom Order

Customize the order and visibility of items.

## Usage

### Status Bar

1. Click the list icon in the ribbon or run the command
2. Toggle drag mode to reorder items
3. Use checkboxes to show/hide items
4. Drag rows in the menu or items in the status bar
